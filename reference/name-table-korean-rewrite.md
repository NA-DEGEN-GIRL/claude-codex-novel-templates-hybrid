# Korean Name Table Rewrite

> 한국어 독자 기준으로 바로 활용할 수 있도록 다시 짠 이름 참고표.
> 핵심 원칙: **본명 / 별호 / 법명 / 현대 실명 / 성씨 조합**을 분리한다.
> 무협/사극에서 뜻이 너무 직접적인 이름은 본명보다 별호나 무공명으로 우선 검토한다.

## How to Use
- 먼저 `용도`를 고른다: 본명 / 별호 / 법명 / 현대 실명 / 성씨 조합
- 다음 `시대`를 고른다: 고대/무협 / 현대
- 마지막으로 `태그`로 인상을 조정한다: 차가움, 강함, 고귀함, 순수함, 퇴폐적, 신비로움

| 성씨 | 인상 | 잘 붙는 이름 리듬 | 예시 조합 |
|------|------|------------------|----------|
| 김 | 가장 무난함 | 거의 전부 무난 | 김무현, 김서연 |
| 이 | 단정하고 보편적 | 유려한 이름과 잘 맞음 | 이도윤, 이명월 |
| 박 | 단단하고 현실적 | 강한 초성 이름과 잘 맞음 | 박한소, 박민준 |
| 최 | 날렵하고 선명함 | 이음절 이름과 궁합 좋음 | 최서경, 최하람 |
| 정 | 부드럽고 고전적 | 유음 계열과 잘 맞음 | 정연화, 정온유 |
| 강 | 강한 인상 | 무협/사극 이름에 특히 잘 맞음 | 강무현, 강적하 |
| 윤 | 세련되고 맑음 | 여성형/중성형에 잘 맞음 | 윤설향, 윤서율 |
| 한 | 고전적이고 또렷함 | 무협/사극 이름과 궁합 좋음 | 한청산, 한명월 |
| 진 | 무게감 있음 | 무협 이름과 잘 맞음 | 진묵연, 진매설 |
| 서 | 서늘하고 문기 있음 | 지적/차가운 인상과 잘 맞음 | 서경, 서도연 |

### 성씨 운용 원칙
- 무협/사극에서 너무 희귀한 성씨를 남발하지 않는다.
- 이름 단독보다 `성+이름`으로 입에 붙는지 먼저 확인한다.
- 주인공만 과하게 특이하게 만들지 말고, 주변 인물과의 대비로 튀게 한다.

## 10. 한 줄 원칙

- **뜻이 너무 선명하면 본명보다 별호일 가능성이 높다.**
- **한국어 이름은 한자 뜻보다 실제 입에 붙는 리듬이 우선이다.**
- **무협 본명은 은근해야 하고, 별호는 과장되어도 된다.**
- **현대 실명은 '희귀함'보다 '실재감'이 더 중요하다.**
